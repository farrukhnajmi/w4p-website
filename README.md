# Running Website Locally
1. pushd /home/farrukh/wellfleet4palestine/w4p-website
2. hugo server -D &
3. Open in browser: http://localhost:1313/

# Publishing Website To GCloud Storage
1. Generate website to public folder: `hugo`
2. Copy website to gcloud: `gsutil rsync -R /home/farrukh/wellfleet4palestine/w4p-website/public gs://www.wellfleet4palestine.org`

# gcloud Vs gsutil
[The gcloud CLI is newer and more comprehensive](https://stackoverflow.com/a/58147001). Use it when possible.
